# Välkommen!

Detta är ett exempelinlägg som visar hur du kan strukturera dina blogginlägg.

## Markdown-formatering

Du kan använda all standard markdown-formatering:

- **Fetstil**
- *Kursiv*
- `Kod`
- [Länkar](https://example.com)

### Kodblock

```javascript
function hello() {
  console.log("Hej världen!");
}
```

## Bilder

Lägg till bilder genom att referera till dem från `/public`-mappen eller använd externa URLs.

## Slutsats

Börja skriva ditt eget innehåll genom att skapa nya `.md`-filer i `src/content/blog/`-mappen.
